This demo just an example to show you how to develop a WeiXin mini program.


## Structure

```console

.
├── README.md
├── app.js    // required
├── app.json  // optional
├── app.wxss  // optional
├── pages     
│   ├── home
│   │   ├── home.js      // required
│   │   ├── home.json    // optional
│   │   ├── home.wxml    // required
│   │   └── home.wxss    // optional
│   └── second
│       ├── second.js    // required
│       └── second.wxml  // required
├── project.config.json
└── sitemap.json

```
